# 迷宫游戏

一个使用 HTML5 Canvas 和 JavaScript 制作的随机迷宫游戏。

## 游戏说明

- 使用方向键或 WASD 移动黑色角色
- 从左上角移动到右下角的黑色方块即可获胜
- 每次获胜后会自动生成新的随机迷宫
- 第3关开始会出现旋转陷阱
- 支持移动端触摸控制
